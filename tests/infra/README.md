# Welcome to your CDK TypeScript project!

This is a blank project for TypeScript development with CDK.

The `cdk.json` file tells the CDK Toolkit how to execute your app.

## Useful commands

- `pnpm build`                     compile typescript to js
- `pnpm watch`                     watch for changes and compile
- `pnpm test`                      perform the jest unit tests
- `pnx aws-cdk@latest deploy`      deploy this stack to your default AWS account/region
- `pnx aws-cdk@latest diff`        compare deployed stack with current state
- `pnx aws-cdk@latest synth`       emits the synthesized CloudFormation template
